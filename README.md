# Spin Wheel / EN
A customizable fortune wheel for random selections.

## Links
* Try it: **https://fredericwithc.github.io/roda-roleta/**
* Repository: **https://github.com/fredericwithc/roda-roleta**

## About
Interactive web application that allows users to create a fully customizable fortune wheel. Features real-time editing, intelligent distribution, and precise result detection.

## Features
* Dynamic option management (add/edit/remove up to 8 options)
* Color picker for each option
* Smart distribution algorithm (avoids adjacent duplicates)
* Automatic text resizing based on content length
* Real-time distribution statistics
* Smooth spin animations with accurate detection
* Fully responsive design (mobile/tablet/desktop)

## Technologies
* HTML5
* CSS3 
* JavaScript

## Key Learnings
* Creating geometric shapes with CSS clip-path
* Building distribution algorithms for balanced results
* Implementing responsive design with 4 breakpoints
* Managing application state reactively
* Synchronizing CSS animations with JavaScript logic
* Dynamic font sizing based on text length

# Roda Roleta / PT-BR
Uma roleta da fortuna customizável para seleções aleatórias.
## Links
* Experimente: **https://fredericwithc.github.io/roda-roleta/**
* Repositório: **https://github.com/fredericwithc/roda-roleta**

## Sobre
Aplicação web interativa que permite aos usuários criar uma roleta da fortuna totalmente personalizável. Apresenta edição em tempo real, distribuição inteligente e detecção precisa de resultados.

## Funcionalidades
* Gerenciamento dinâmico de opções (adicionar/editar/remover até 8 opções)
* Seletor de cor para cada opção
* Algoritmo de distribuição inteligente (evita duplicatas adjacentes)
* Redimensionamento automático de texto baseado no comprimento
* Estatísticas de distribuição em tempo real
* Animações suaves de giro com detecção precisa
* Design totalmente responsivo (mobile/tablet/desktop)

## Tecnologias
* HTML5
* CSS3
* JavaScript

## Principais Aprendizados
* Criação de formas geométricas com CSS clip-path
* Construção de algoritmos de distribuição para resultados balanceados
* Implementação de design responsivo com 4 breakpoints
* Gerenciamento de estado da aplicação de forma reativa
* Sincronização de animações CSS com lógica JavaScript
* Dimensionamento dinâmico de fonte baseado no comprimento do texto
